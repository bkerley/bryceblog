iCal annoyed me with frequent error messages about Google’s janky-ass calendar service being occasionally unreachable. On occasion, these messages would block the UI in such a way that required me to force-quit iCal.

{% endraw %} {% raw %}

BusyCal doesn’t! I’m willing to overlook some UI strangeness (goofy panels, fonts that aren’t pixel-perfect to iCal) for this.

{% endraw %} {% raw %}

Try it for 30 days perhaps!

{% endraw %} {% raw %}
